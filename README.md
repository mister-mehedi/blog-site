# blog-site
## RESTfull Blog Api

- Designed and developed a secure backend solution for a modern, multi-user content platform, structuring the application with the MVC design pattern.
- Implemented a robust security system featuring JSON Web Token authentication with refresh tokens and a multi-level Role-Based Access Control (RBAC) mode manage permissions across four distinct user roles.
- Engineered a complete content moderation workflow, enabling full CRUD operations for users and posts, and establishing an admin approval process for content submitted by clients to ensure quality control.
